This page outlines the geographic availability, supported asset classes, and settlement methods available across the Zelkea network.

---

## 🌍 Supported countries

Zelkea can support networks, institutions, and platforms operating in the following regions.

| Status | Details |
| :-- | :-- |
| **Available** | Global availability across supported institutions except restricted regions. |
| **Unavailable** | Services are currently unavailable for entities located in Afghanistan, Belarus, Central African Republic, Cuba, Democratic Republic of the Congo, Iran, Iraq, Libya, Myanmar, North Korea, Russia, Somalia, South Sudan, Sudan, Syria, and Venezuela. |

---

## 🪙 Supported currencies

Zelkea separates capabilities by handle registration and transaction execution.

| Currency | Financial handles | Transactions (Settlement) |
| :-- | :-: | :-: |
| **USD** (US Dollar) | ✅ Supported | ✅ Supported |
| **EUR** (Euro) | ❌ Unsupported | ✅ Supported |

---

## 🏦 Deposits & withdrawals

Institutions can manage their pre-funded network balances using the following settlement rails.

| Method | Type |
| :-- | :-- |
| **ACH** | Domestic US transfers |
| **SEPA** | Eurozone transfers |
| **Wire** | Real-time gross settlement |
| **SWIFT** | Cross-border institutional settlement |
