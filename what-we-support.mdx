## Supported countries

Zelkea supports networks, institutions, and platforms operating in the following regions.

| Status | Details |
| :-- | :-- |
| **Available** | Zelkea is available to legal entities that are not headquartered in restricted regions. |
| **Unavailable** | Zelkea is currently unavailable for entities located in Afghanistan, Belarus, Central African Republic, Cuba, Democratic Republic of the Congo, Iran, Iraq, Libya, Myanmar, North Korea, Russia, Somalia, South Sudan, Sudan, Syria, and Venezuela. |

---

## Supported currencies

We are continuously expanding currency support based on ecosystem demand.

| Currency | Handles | Transactions |
| :-- | :-: | :-: |
| **USD** (US Dollar) | Supported | Supported |
| **EUR** (Euro) | Unsupported | Supported |

## Deposits and withdrawals

The following settlement methods are available for institutional network balances:

| Type | Rails |
| :-- | :-- |
| **Deposits** | ACH, WIRE, SEPA, SWIFT |
| **Withdrawals** | ACH, WIRE, SEPA, SWIFT |
