---
title: "Make your first API call"
---

This guide helps you set up your institution account, issue your first customer payment handle, and execute an instant network transfer.

---

## 1. Sign up your institution

Before using the API, your institution must register its top-level routing domain.

- Go to [https://app.zelkea.com](https://app.zelkea.com) and create an institution account.
- Complete onboarding and retrieve your API secret key.

```bash
export ZELKEA_API_KEY="your_api_key_here"
```

## 2. Create a payment handle

Every user on the network needs a unique payment handle (`user@domain`).

<RequestExample>

```bash title="Request"
curl -X POST https://api.zelkea.com/handles \
  -H "Authorization: Bearer $ZELKEA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "handle": "alice",
    "balance": 1000.00,
    "currency": "USD",
    "limit": 5000.00,
    "code": "auth-secure-1234",
    "status": "active"
  }'
```

</RequestExample>

<ResponseExample>

```json title="Response"
{
  "success": true,
  "data": {
    "handle": "alice@acme",
    "balance": 1000,
    "currency": "USD",
    "limit": 5000,
    "status": "active",
    "code": "auth-secure-1234",
    "created_at": "2026-07-10T00:21:49Z"
  }
}
```

</ResponseExample>

## 3. Execute a transfer

Move funds instantly across the network.

<RequestExample>

```bash title="Request"
curl -X POST https://api.zelkea.com/transfers \
  -H "Authorization: Bearer $ZELKEA_API_KEY" \
  -H "Idempotency-Key: 7b9e4a3c-2d1e-4f5a-8b6c-9d8e7f6a5b4c" \
  -H "Content-Type: application/json" \
  -d '{
    "sender": "alice@acme",
    "receiver": "bob@chase",
    "amount": 250,
    "currency": "USD"
  }'
```

</RequestExample>

<ResponseExample>

```json title="Response"
{
  "success": true,
  "data": {
    "id": "tx_987654321",
    "sender": "alice@acme",
    "receiver": "bob@chase",
    "amount": 250,
    "currency": "USD",
    "created_at": "2026-07-10T00:22:00Z"
  }
}
```

</ResponseExample>
