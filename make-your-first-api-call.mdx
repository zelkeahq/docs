---
title: "Make your first api call"
---

This guide helps you set up your institution account, issue your first customer payment handle, and execute an instant network transfer.

---

## 1. Sign up your institution

Before using the API, your institution must register its top-level routing domain.

- Go to [://zelkea.com](https://://zelkea.com) and create an institution account.
- Complete your onboarding and retrieve your API secret key from your dashboard.

Set your key as an environment variable to authenticate requests:

```bash
export ZELKEA_API_KEY="your_api_key_here"
```

## 2. Create a payment handle

Every user on the network needs a unique payment handle (`user@domain`) to route transactions securely. The domain suffix is appended automatically based on your institution configuration.

<CodeGroup>

```bash cURL Request
curl --location --request POST 'https://zelkea.com' \
--header 'Authorization: Bearer \$ZELKEA_API_KEY' \
--header 'Content-Type: application/json' \
--data-raw '{
  "handle": "alice",
  "balance": 1000.00,
  "currency": "USD",
  "limit": 5000.00,
  "code": "auth-secure-1234",
  "status": "active"
}'
```

```json Response
{
  "success": true,
  "data": {
    "handle": "alice@acme",
    "balance": 1000.00,
    "currency": "USD",
    "limit": 5000.00,
    "status": "active",
    "code": "auth-secure-1234",
    "created_at": "2026-07-10T00:21:49Z"
  }
}
```

</CodeGroup>

## 3. Execute a transfer

Move funds instantly across the network from an authorized sender handle to a receiver handle. You must include a unique UUID as an `idempotency-key` header to prevent double-processing.

<CodeGroup>

```bash cURL Request
curl --location --request POST 'https://zelkea.com' \
--header 'Authorization: Bearer \$ZELKEA_API_KEY' \
--header 'idempotency-key: 7b9e4a3c-2d1e-4f5a-8b6c-9d8e7f6a5b4c' \
--header 'Content-Type: application/json' \
--data-raw '{
  "sender": "alice@acme",
  "receiver": "bob@chase",
  "amount": 250.00,
  "currency": "USD"
}'
```

```json Response
{
  "success": true,
  "data": {
    "id": "tx_987654321",
    "sender": "alice@acme",
    "receiver": "bob@chase",
    "amount": 250.00,
    "currency": "USD",
    "created_at": "2026-07-10T00:22:00Z"
  }
}
```

</CodeGroup>
