Zelkea SDKs make integration faster by handling authentication, request formatting, and API communication for you. Instead of building directly against the API, use our official client libraries to get started with less boilerplate.

We currently provide an official JavaScript SDK. Additional SDKs are planned — if you need support for another language, contact [support@zelkea.com](mailto:support@zelkea.com).

## JavaScript

Install the SDK:

```bash
npm install @zelkea/node-sdk
```

Initialize the client with your API key:

```javascript
import { Zelkea } from "@zelkea/node-sdk";

const zelkea = new Zelkea({
  apiKey: "your_api_key"
});
```
